# Ravine

A split hummingbird-esque keyboard, inspired by ezxzeng's sweep_squared_rot.
Notable changes:
  - Completely modified layout,
  - Charlieplex matrix (with interrupt lane),
  - Compatibility with a 2032 coin cell,
  - Xiao MCU.
